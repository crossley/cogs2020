Effect Size

  • How big is an effect?

  • The answer depends on the context and the statistical test being performed.

  • In a single-sample test, where we compare a sample mean to a hypothesized value (e.g., \(\mu_0\)), we can use a standardized difference between them.

  • This gives us a way to assess whether a result is not just statistically significant, but meaningful.

Why Use the Raw Data Distribution?

  • Effect size is computed in the space of the raw data, not the test statistic.

  • It answers the question: How far is the observed mean from the hypothesized mean, in meaningful units?

  • The raw data distribution reflects real-world variability in your measurements.

  • In contrast, the test statistic distribution (e.g., Normal or t) reflects variability due to sampling.

  • Use test statistic distribution for:

    • Hypothesis testing (NHST decisions)
    • Power analysis (detectability under \(H_1\))

  • Use raw data distribution for:

    • Effect size
    • Power analysis (to define the size of the true effect)

  • Both are needed for power: the effect size tells you how far \(\bar{x}\) is from \(\mu_0\), and the test stat tells you how likely it is to be detected.

Cohen’s d (Single-Sample Version)

  • Cohen’s d is a widely used measure of effect size.

  • For a single-sample test (with known population standard deviation), it is:

\[ d = \frac{\bar{x} - \mu_0}{\sigma} \]

  • This tells us how far the sample mean is from the hypothesized mean, in standard deviation units.

  • For example, if \(\bar{x} = 1\), \(\mu_0 = 0\), and \(\sigma = 1\), then \(d = 1.0\).

Key Takeaway

  • Effect size helps answer: Is the result meaningful, not just significant?

  • For single-sample tests, it’s simply the standardized difference from the hypothesized value.

  • When we later talk about comparing two groups, we’ll extend this idea using a different version of Cohen’s d.
